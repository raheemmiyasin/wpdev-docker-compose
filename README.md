# WPDEV Docker Compose

docker-compose.yml to help wordpress developer run development wordpress using Docker container.

## Requirements
- Docker
- docker-compose

## Installation

Clone project


```bash
git clone
cd wpdev-docker-compose
```

## Usage

```bash
docker compose up
```
